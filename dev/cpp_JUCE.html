JUCE library MinGW win64 port (C/C++, VST2, GLEW, many other libs)

Ported JUCE to MinGW compiler suite on Windows64.
No namespace pollution with Windows headers
Includes several VST Plugins and standalone apps to test the JUCE port
(work in progress) fftw3f is included precompiled, not by source (yet)
(work in progress) I disabled libCurl because its dependencies libssl and libcrypto crashed the debugger!

Features:

  • Produces sound using WASAPI, Steinberg ASIO and DirectSound.
  • Advanced MIDI (MPE - Midi polyphonic expressions)
  • GUI system, DSP, Desktop OpenGL.
  • Includes minimal_vst2_host_x64 app for testing VSTi 2.4 Plugin dlls. ( exit app with ESCAPE key )
  • Includes file2header app, converts any file into embedded .h resource header file to be compiled in ( saves .h next to input file ).
  • Includes ObXd synthesizer vst2 x64 (standalone.exe and plugin.dll).
  • Includes OPL synthesizer vst2 x64 (standalone.exe and plugin.dll).
  • Includes Synister synthesizer vst2 x64 (standalone.exe and plugin.dll).
  • Includes TapSynth synthesizer vst2 x64 (standalone.exe and plugin.dll).
  • Includes Manta vst2 x64 (standalone.exe and plugin.dll). Not sure if its a synth or effect
  • Includes many libs to make JUCE compile
  • Other open source JUCE projects will follow
