AM Synth Win64 port (C++, Qt, RtAudio, RtMidi)

I like this synth, so I ported it to Windows64 using Qt, RtAudio, RtMidi and Steinberg ASIO.
Port is not complete, only MIDI keyboard keys yet, no CC, no Pedals, no gui scaling yet.
Use your computer keyboard to play some melodies. Many keys are mapped to midi notes
Beware that leaving window with your mouse can lead to missing key release events

Features:

  • Produces sound using RtAudio, default is WASAPI, Steinberg ASIO possible, needs config change and recompile (yet).
  • Preset loading and selection is working now)
  • The MIDI keys of my Digital Piano (YAMAHA P-105) are working, not the Sustain Pedal or Buttons, yet.
  • Computer keyboard produces MIDI messages. MainWindow needs focus for key-events, best click on Audition Button (to get focus) and then use the keyboard.
  • Panic Button is working.
  • GUI uses QtWidgets, ported from GTK which was a C mess
  • RtMidi is implemented rudimentary.
  • Uses std::filesystem ( C++17 feature ).
