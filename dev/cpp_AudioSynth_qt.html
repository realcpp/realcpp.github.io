  • QtSDK( GUI, C/C++ compiler, debugger, mingw32-make, QtCreator )

    • develop C++ on Windows using only free software
    • The integrated MinGW GNU C++ 7.3.1 (c++17) compiler/debugger is my daily used compiler to develop on Windows64
    • I always install QtSDK and use integrated MinGW GNU C++ compiler/debugger + QtCreator as daily IDE
  • RtAudio ( audio engine backend, includes Steinberg ASIO SDK )
  • FFMPEG-shared-release-libs ( loads audio from multimedia containers )
  • OpenGLES + EGL + ANGLE ( 3D oscilloscope )
  • Steinberg VST2SDK ( loading x64 VST24 plugins )
  • DarkImage ( my image lib )
  • taglib-1.11.1 ( loading coverarts )
  • SoundTouch ( time stretch and pitch shift used in audio file player with FFMPEG )
  • KissFFT + fftw3 ( Fast Fourier Transform for Spectrograph )
